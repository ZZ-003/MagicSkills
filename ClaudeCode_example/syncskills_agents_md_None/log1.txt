  AST基础知识                                                                                        
   
  抽象语法树（Abstract Syntax Tree, AST） 是程序语法结构的层次化表示，具有以下特点：                 
  - 表示语法结构，而非运行时行为                                                                   
  - 忽略格式、注释和空白字符                                                                         
  - 编码程序的声明、语句和表达式结构                                                               
                                                                                                     
  C语言AST的核心节点类型                                                                             
                                                                                                     
  1. 根节点                                                                                          
                                                                                                     
  - FileAST: 整个翻译单元的根节点，包含所有全局声明和函数定义                                        
                                                                                                   
  2. 函数相关                                                                                        
                                                                                                   
  - FuncDef: 函数定义，包含函数声明、参数列表和函数体                                                
  - Decl: 声明节点，用于变量、函数、参数等
                                                                                                     
  3. 类型系统                                                                                        
                                                                                                     
  - TypeDecl: 基本类型声明（如 int, char）                                                           
  - PtrDecl: 指针类型声明
  - ArrayDecl: 数组类型声明                                                                          
                                                                                                     
  4. 语句节点                                                                                        
                                                                                                     
  - Compound: 复合语句块 {...}                                                                       
  - Assignment: 赋值语句
  - Return: 返回语句                                                                                 
  - If/While/For: 控制流语句                                                                         
                                                                                                     
  5. 表达式节点                                                                                      
                                                                                                     
  - ID: 变量引用                                                                                     
  - Constant: 字面量常量（整数、字符串等）
  - BinaryOp: 二元操作（+, -, *, /, ==, && 等）                                                      
  - FuncCall: 函数调用                                                                               
                                                                                                     
  实际应用价值                                                                                       
                                                                                                     
  AST在以下领域非常重要：                                                                            
  - 编译器开发: 作为前端到后端的中间表示
  - 静态分析: 检测代码质量问题和安全漏洞                                                             
  - 安全审计: 识别危险函数调用（如 gets, strcpy）
  - 程序理解: 分析代码结构和依赖关系                                                                 
  - 自动化重构: 安全地修改代码结构                                                                   
                                                                                                     
  使用示例                                                                                           
                                                                                                     
  如果你有C代码需要分析，我可以使用 c_2_ast 技能将其转换为AST。例如：

      printf("Hello, World!");                                                                       
      return 0;                                                                                      
  }

    FuncDef:                                                                                         
      Decl: main                                                                                     
      FuncDecl: 
        TypeDecl:                                                                                    
          IdentifierType: ['int']                                                                    
      Compound:                                                                                      
        FuncCall:                                                                                    
          ID: printf                                                                                 
          ExprList:                                                                                  
            Constant: string, "Hello, World!"                                                        
        Return:                                                                                      
          Constant: int, 0
